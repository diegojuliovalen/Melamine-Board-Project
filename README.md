# Melamine Board Data Project

This project was developed for **M.A.D. Center** to analyze and forecast the demand for their melamine boards. It includes data analysis, predictive modeling, and an interactive dashboard.

## Project Structure

- **mad_center_data/**  
  Cleaned dataset used for the analyses and model.

- **analisis_24/**  
  Pandas analysis for the year **2024**.

- **analisis_25/**  
  Pandas analysis for the year **2025**.

- **analisis_2425/**  
  Combined analysis for **2024 and 2025**.

- **Final Model Forecasting Demand/**  
  Random Forest model to predict melamine board demand.

- **mad_center_dashboard/**  
  Interactive dashboard in **Power BI** and exported **PDF**.

## Summary

- Performed exploratory data analysis (EDA) using Pandas for each year and combined data.  
- Built a **Random Forest model** for demand forecasting.  
- Developed an **interactive Power BI dashboard** for insights visualization.  

## Usage

1. Explore the data and analyses in the respective folders.  
2. Use the Random Forest model notebook to understand the demand prediction process.  
3. Open the Power BI dashboard to interact with visual insights.
